# automation
Miscellaneous scripts for task automation

Batch jpeg:
Convert images to webp format (using Irfanview), and compress them in rar (with winrar).

Decrypter:
Decrypts files of following engines: Kikiriki, rpg maker, wolf engine, renpy.

Renamer:
Sort files in folder, and renames them (with a sequence) with parent folder name.

You can add a menu in the context menu to run those scripts. Example: https://medium.com/analytics-vidhya/creating-cascading-context-menus-with-the-windows-10-registry-f1cf3cd8398f
